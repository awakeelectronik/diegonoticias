# Código de Conducta

Este proyecto adopta el Contributor Covenant 2.1.

## Nuestro compromiso
Nos comprometemos a una comunidad abierta, respetuosa e inclusiva.

## Estándares
- Respeto en discusiones técnicas.
- Cero acoso o ataques personales.
- Feedback orientado a mejorar el producto.

## Aplicación
Reporta incidentes al mantenedor principal del repositorio.
